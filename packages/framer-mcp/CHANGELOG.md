# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Framer API coverage
- 20 MCP tools for project management, collections, pages, code files, and publishing
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Project management tools for Framer site configuration
- CMS collection CRUD with field schema introspection
- Page listing, creation, and content management
- Code file operations for custom component development
- One-click publishing and deployment controls
