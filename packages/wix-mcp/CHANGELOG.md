# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Wix API coverage
- 21 MCP tools for data collections, eCommerce, blog, contacts, bookings, and orders
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Data collection management for structured content
- eCommerce product and order tools
- Blog post creation and management
- Contact and CRM management tools
- Bookings and scheduling operations
