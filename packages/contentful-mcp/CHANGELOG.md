# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Contentful API coverage
- 20 MCP tools for content types, entries, assets, environments, locales, tags, and bulk publishing
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Content type and entry management with localization support
- Asset upload and processing across environments
- Environment management for staging and production workflows
- Locale and tag organization tools
- Bulk publishing operations for efficient content deployment
