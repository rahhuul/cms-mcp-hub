# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Yoast SEO API coverage
- 18 MCP tools for SEO metadata, scores, redirects, social data, schema markup, and sitemaps
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- SEO metadata management for posts and pages
- SEO score analysis and readability checks
- Redirect creation and management
- Social media metadata (OpenGraph and Twitter cards)
- Schema markup configuration and sitemap management
