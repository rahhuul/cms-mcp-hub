# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Payload CMS API coverage
- 21 MCP tools for collections, entries, globals, media, access control, version management, and bulk operations
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Collection and entry CRUD with deep filtering
- Global document management for site-wide settings
- Media upload and asset management
- Access control and permission introspection
- Version management and bulk operations for batch workflows
