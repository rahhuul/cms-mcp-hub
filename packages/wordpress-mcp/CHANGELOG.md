# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full WordPress API coverage
- 169 MCP tools for posts, pages, media, comments, taxonomy, users, menus, blocks, plugins, themes, widgets, fonts, site-editor, and workflows
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Complete post and page management with revision history
- Gutenberg block editor integration with component library
- Media library management with upload and metadata editing
- Taxonomy, menu, and widget configuration tools
- Plugin, theme, and site-editor management with webhook listener support
