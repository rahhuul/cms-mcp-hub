# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Ghost API coverage
- 24 MCP tools for posts, pages, tags, authors, members, newsletters, tiers, images, and webhooks
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Post and page management with rich content editing
- Tag and author organization tools
- Member and newsletter management for audience engagement
- Tier configuration for paid subscriptions
- Image upload and webhook integration
