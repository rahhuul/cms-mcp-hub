# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Sanity API coverage
- 16 MCP tools for GROQ queries, documents, assets, datasets, publishing, and transactions
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- GROQ query execution for flexible document retrieval
- Document CRUD with draft and published state management
- Asset upload and management for images and files
- Dataset management and configuration
- Transaction support for atomic multi-document operations
