# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Webflow API coverage
- 21 MCP tools for sites, collections, items, pages, products, orders, publishing, and domains
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Site and domain management tools
- CMS collection and item CRUD operations
- Page listing and content management
- E-commerce product and order management
- One-click site publishing and deployment
