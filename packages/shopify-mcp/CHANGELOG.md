# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Shopify API coverage
- 147 MCP tools for products, collections, orders, customers, inventory, content, and store management
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Complete product and collection management with variants and images
- Order processing, fulfillment, and refund workflows
- Customer management with address and metafield support
- Inventory tracking across multiple locations
- Store configuration and content management tools
