# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full WooCommerce API coverage
- 95 MCP tools for products, orders, customers, coupons, reports, tax, shipping, webhooks, batch operations, and workflows
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Complete product management with variations, attributes, categories, and tags
- Order processing workflows including refunds and order notes
- Customer management with download tracking
- Coupon creation and management
- Tax, shipping, and webhook configuration
