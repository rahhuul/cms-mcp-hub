# Changelog

All notable changes to this package will be documented in this file.

## [0.3.0] - 2025-05-15

### Added
- Initial release with full Strapi API coverage
- 17 MCP tools for content types, entries, media, users, roles, and i18n localization
- API client with retry logic and rate limiting
- Zod schema validation for all tool inputs
- Error handling with actionable suggestions

### Features
- Content type introspection and schema discovery
- Entry CRUD with filtering, sorting, and pagination
- Media library upload and management
- User and role management with access control
- i18n localization support for multilingual content
